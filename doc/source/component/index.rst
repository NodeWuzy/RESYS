=================================
水力学部件
=================================

在RESYS中，水力学部件用于划分流动控制体以及指定特殊模型。
本章节讨论各个水力学部件的模型的细节和特点。
而这些模型是建立反应堆系统模型的基础。
不正确的理解这些模型可能会导致最终建立的反应堆冷却剂回路模型与所想要建立的模型相差甚远。

因此要正确的开发或者使用RESYE程序，需要仔细阅读本章节。
