=================================
守恒方程的离散
=================================

质量、动量、能量守恒方程的数值求解依赖于将这些方程进行离散。对这些方程进行合理的离散化是求解这些方程的基础。
在这一章中，我们会讨论如何离散各种守恒方程。
首先我们会讨论如何在交错网格上离散守恒方程中的每一项，这会在 **方程离散的总体思路** 这一小节中。
然后讨论网格数据的划分与生成，这对应于 **方程离散的网格的划分** 这一小节。
之后讨论稀疏矩阵的产生，这对应于 **矩阵与方程** 这一小节。
再然后讨论流场的插值，这对应于 **流场与插值** 这一小节。

最后针对各种类型的守恒方程，具体的给出其离散的形式。
